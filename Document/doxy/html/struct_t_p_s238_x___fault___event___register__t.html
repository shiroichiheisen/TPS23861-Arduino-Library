Detailed Description

Fault Event Register - Software API uses TPS238x_Ports_t; Software retrieves these events as part of tps_GetAllInterruptEvents()
Hardware - Command 06h; 1 data byte; read only

  • Command 07h; 1 data byte; Clear on read

    ICUT fault and disconnect events occurred for at least one port
