Detailed Description

Legacy Detect Mode Register - Software API uses TPS238x_Ports_t in tps_ConfigPort()
Hardware - Command 20h; 1 data byte; read/write

The TPS23861 can perform a legacy-detect operation only or a standard detection followed by a legacy-detect operation.
